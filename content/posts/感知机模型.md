本文介绍了最基础的感知机模型，并给出了其 python 简单实现.

<!--more-->

## 概述

当输入空间为 $\mathbb R^n$ 而输出空间为 $\{+1,-1\}$ 时，本质上是要做一个二分类的任务，感知机模型适用于数据集线性可分的简单情况，具体而言，我们的预测函数定义为
$$
f(x)=\operatorname{sgn}(\boldsymbol w \cdot \boldsymbol x + b)
$$
其中 $\boldsymbol w,b$ 为感知机模型参数，$\boldsymbol w \in \mathbb R^n$ 被称作权值（向量），$b \in \mathbb R$ 被称作偏置. $\boldsymbol w \cdot \boldsymbol x + b = 0$ 这个超平面将空间分为两部分，因此如果数据集线性可分，那么必然可以找到一个合理的超平面将输出为 $+1$ 和 $-1$ 的输入恰好分开.

例如，AND、OR 是线性可分的，但 XOR 并不是线性可分的.

## 感知机

我们需要定义一个合理的损失函数并将其最小化，一个可行的定义是所有误分类数据到超平面的距离之和，点 $\boldsymbol x_0$ 到超平面 $\boldsymbol w \boldsymbol x + b=0$ 的距离为
$$
\frac{|\boldsymbol w\cdot\boldsymbol x_0 + b|}{\|\boldsymbol w\|}
$$
因为 $\boldsymbol x_0$ 被错分，因此 $y_i(\boldsymbol w\boldsymbol x_0+b)<0$，而 $|y_i|=1$，故此时距离就是
$$
-\frac{1}{\|\boldsymbol w\|}y_i(\boldsymbol w\cdot\boldsymbol x_0+b)
$$
设被误分的点集为 $\mathcal S(\boldsymbol w)$，其总距离就是
$$
-\frac{1}{\|\boldsymbol w\|}\sum_{\boldsymbol x_i\in\mathcal S}y_i(\boldsymbol w\cdot\boldsymbol x_i+b)
$$
我们的目标是使上式为 $0$，找到一组可行的 $\boldsymbol w,b$，$\frac{1}{\|\boldsymbol w\|}$ 不改变上式是否为 $0$，去掉它得到损失函数
$$
L(\boldsymbol w,b)=-\sum_{\boldsymbol x_i \in \mathcal S(\boldsymbol w)} y_i(\boldsymbol w \cdot \boldsymbol x_i + b)
$$
> ![NOTE]
>
> 这里可以直接去掉 $\frac{1}{\|\boldsymbol w\|}$ 的原因是为了在不应该正确性的同时简化计算，不带 $\frac{1}{\|\boldsymbol w\|}$ 仍然可以评估点到超平面的距离，只要存在误分类点，梯度下降就会继续.

上式 $\ge 0$，要求其最小值点，求一下梯度
$$
\nabla_{\boldsymbol w} L =-\sum_{\boldsymbol x_i \in \mathcal S(\boldsymbol w)}y_i\boldsymbol x_i
$$

$$
\nabla_b L = -\sum_{\boldsymbol x_i \in \mathcal S(\boldsymbol w)}y_i
$$

因此只需选取一个误分类点 $(\boldsymbol x_i,y_i)$ 并做更新
$$
\begin{aligned}
\boldsymbol w \leftarrow w + \eta y_i\boldsymbol x_i\\
b \leftarrow b + \eta y_i
\end{aligned}
$$
直至没有误分类点.

其实这里还有一个隐蔽的问题，就是我们并没有用梯度更新，而是每次仅根据一个误分类点更新，不过这样仍然可以收敛到一个正确的超平面，这是由感知机收敛性定理保证的，这里不作证明.

## 对偶形式

根据我们的训练方式，可以得知
$$
\begin{aligned}
\boldsymbol w = \sum_{i=1}^{n}\alpha_i y_i\boldsymbol x_i\\
b=\sum_{i=1}^{n} \alpha_i y_i
\end{aligned}
$$
我们为什么不能直接学习 $\boldsymbol \alpha$？这是一个合理的想法，对应算法流程如下：

- $\boldsymbol \alpha\leftarrow \boldsymbol 0,b\leftarrow 0$
- 在训练集中选取 $(\boldsymbol x_i,y_i)$
- 若 $y_i(\sum_{j=1}^{n}\alpha_jy_j\boldsymbol x_j \cdot \boldsymbol x_i + b) \le 0$，则更新

$$
\begin{aligned}
\alpha_i \leftarrow \alpha_i + \eta\\
b \leftarrow b+\eta y_i
\end{aligned}
$$

- 直到没有误分类数据.

实现时，需要预处理所有的 $\boldsymbol x_i \cdot \boldsymbol x_j$，也就是 Gram矩阵，之后我们就不必每次计算点积，后续的迭代速度只和样本数量相关.

## 代码实现

```python {group = "my_code",.line-wrapping, name="感知机", title="感知机"}
# %%
import numpy as np

# %%
class Perceptron(object):
    def __init__(self):
        self.w = None
        self.bias = None
        self.eta = 0.1
    def train(self, x_train, y_train):
        x_train = np.array(x_train, dtype = float)
        y_train = np.array(y_train, dtype = float)
        self.w = np.zeros(x_train.shape[1], dtype = float)
        self.bias = 0.0
        while True: # 需要保证训练集线性可分，不然会死循环.
            # 也可以增加一个 max_iter 控制迭代次数
            index = 0
            while index < x_train.shape[0]:
                y_predict = self.predict(x_train[index])
                if y_predict != y_train[index]:
                    self.w += self.eta * y_train[index] * x_train[index]
                    self.bias += self.eta * y_train[index]
                    break
                index += 1
            if index == x_train.shape[0]:
                break
    def predict(self, x):
        return 1 if np.dot(x, self.w) + self.bias > 0 else -1
    def __str__(self):
        return f'weights = {self.w}, bias = {self.bias}'

# %%
def get_training_dataset():
    # 训练数据：输入向量和标签
    input_vecs = [[1,1], [0,0], [1,0], [0,1]]
    labels = [1, -1, -1, -1] # and 函数
    # labels = [-1, 1, 1, 1] # or 函数
    # labels = [-1, -1, 1, 1] # xor 函数，无法用单层感知机学习
    return input_vecs, labels
def train_and_perceptron():
    p = Perceptron()
    input_vecs, labels = get_training_dataset()
    p.train(input_vecs, labels)
    return p

if __name__ == '__main__':
    and_perceptron = train_and_perceptron()
    print(str(and_perceptron))
    for input_vec in [[1,1], [0,0], [1,0], [0,1]]:
        print(f'{input_vec} -> {and_perceptron.predict(input_vec)}')


```

```python {group = "my_code",.line-wrapping, name="感知机的对偶形式", title="感知机的对偶形式"}
# %%
import numpy as np

# %%
class Perceptron_Dual(object):
    def __init__(self):
        self.alpha = None
        self.w = None
        self.bias = None
        self.eta = 0.1
    def train(self, x_train, y_train):
        x_train = np.array(x_train, dtype = float)
        y_train = np.array(y_train, dtype = float)
        self.alpha = np.zeros(x_train.shape[0], dtype = float)
        self.bias = 0.0
        Gram = np.dot(x_train, x_train.T)
        while True: # 需要保证训练集线性可分，不然会死循环.
            # 也可以增加一个 max_iter 控制迭代次数
            for i in range(x_train.shape[0]):
                val = np.dot(self.alpha * y_train, Gram[:, i]) + self.bias
                y_predict = 1 if val > 0 else -1
                if y_predict != y_train[i]:
                    self.alpha[i] += self.eta
                    self.bias += self.eta * y_train[i]
                    break
            else:
                break
        self.w = np.dot(self.alpha * y_train, x_train)
    def predict(self, x):
        return 1 if np.dot(x, self.w) + self.bias > 0 else -1
    def __str__(self):
        return f'alpha = {self.alpha}, weights = {self.w}, bias = {self.bias}'

# %%
def get_training_dataset():
    # 训练数据：输入向量和标签
    input_vecs = [[1,1], [0,0], [1,0], [0,1]]
    # labels = [1, -1, -1, -1] # and 函数
    labels = [-1, 1, 1, 1] # or 函数
    # labels = [-1, -1, 1, 1] # xor 函数，无法用单层感知机学习
    return input_vecs, labels
def train_and_perceptron():
    p = Perceptron_Dual()
    input_vecs, labels = get_training_dataset()
    p.train(input_vecs, labels)
    return p

if __name__ == '__main__':
    and_perceptron = train_and_perceptron()
    print(str(and_perceptron))
    for input_vec in [[1,1], [0,0], [1,0], [0,1]]:
        print(f'{input_vec} -> {and_perceptron.predict(input_vec)}')

```

## 基于 sklearn 的实现

自己编写的代码性能还是不够优秀，可以基于 sklearn 实现感知机.

```python {group="sklearn",.line-wrapping, name="sklearn 感知机实现 AND"}
import numpy as np
from sklearn.linear_model import Perceptron

# sklearn
x_train = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y_train = np.array([0, 0, 0, 1]) # AND gate output
model = Perceptron()
model.fit(x_train, y_train)

x_test = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
predictions = model.predict(x_test)
print(predictions)
print(f'w: {model.coef_}, b: {model.intercept_}')
res = model.score(x_test, y_train)
print(f'Accuracy: {res:.2%}')
```

``` python {group="sklearn",.line-wrapping, name="sklearn 感知机实现随机线性可分数据"}
# %%
import numpy as np
from sklearn.linear_model import Perceptron

# %%
# sklearn Perceptron 的随机线性可分数据实现
w = np.random.uniform(-1, 1, (2,))
b = np.random.uniform(-1, 1)
train_set_size = 1000
x_train = np.random.uniform(-10, 10, (train_set_size, 2))
y_train = np.where(np.dot(x_train, w) + b > 0, 1, -1)
model = Perceptron()
model.fit(x_train, y_train)

x_test = np.random.uniform(-10, 10, (1000, 2))
y_test = np.where(np.dot(x_test, w) + b > 0, 1, -1)
predictions = model.predict(x_test)
accuracy = np.mean(predictions == y_test)
print(f'Accuracy: {accuracy:.2%}')
```
